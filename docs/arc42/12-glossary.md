# 12. Glossary

- ADR: Architectural Decision Record
- arc42: Architecture documentation structure (12 sections)
- Spin: Fermyon Spin, a runtime for WebAssembly components
- WASI: WebAssembly System Interface
- Gateway: the component that centralizes routing/policy and outbound integration
- Service chaining: calling other Spin components via `http://<component-id>.spin.internal`
- HTMX: HTML-over-the-wire progressive enhancement library
